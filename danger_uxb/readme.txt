Danger UXB
++++++++++

V1.0 26/04/04

Sound will come it later versions, I can't be bothered to set up a computer that can record sound :p

Controls

left cursor = left
right cursor = right
up cursor = up
down cursor = down
z or < = shift left
x or > = shift right

For instruntions, run game and press Y when prompted.

Highscores stored in score.txt delete file to reset highscore.

Requires directx 9.0b.
To compile source use dbpro and bluegui
